# VectorLab · Unidad 1: Vectores · Nueva Base V3

Nueva versión limpia y estable de la app.

## Incluye

- Un solo modo de uso.
- 4 secciones principales.
- Navegación: Teoría / Actividades / Autoevaluación.
- Sección 1 desarrollada.
- Tres gráficas interactivas:

  2. Suma de vectores.
